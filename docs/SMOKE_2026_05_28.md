## P1 Fix Applied

`GET /api/storage/analytics` previously used SQLite-only `datetime('now', '-N days')`. Replaced with standard `NOW() - INTERVAL 'N days'` (Postgres). Additionally fixed `decimal.Decimal / float` division error that manifested on Postgres (SQLite returns int, Postgres returns Decimal). Both issues patched in `backend/app/storage/router.py`.
